# Contributing

We welcome contributions that improve the Scrappy Kin codebase and documentation.

## How to contribute
- Fork the repo and open a pull request.
- Use Git sign-off on every commit: `git commit -s`.
- No CLA is required.

## Scope
- **In scope:** code and documentation changes.
- **Out of scope:** research contributions (please email support@scrappykin.com).

## DCO (Developer Certificate of Origin)
By signing off your commits, you certify that you wrote the code or have the right to submit it under this license.
