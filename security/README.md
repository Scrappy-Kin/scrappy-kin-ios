# Security Audits

This directory contains periodic security audit reports for the scrappy-kin-ios codebase.

## Status

No audits published yet. First audit in progress — see [SECURITY.md](../SECURITY.md) for our disclosure stance.

## Format

Audit files are named `YYYY-MM-DD-audit-[scope].md`. Each audit is a point-in-time snapshot. See the repository commit history for ongoing changes between audits.

## Policy

- Audits are published after P0 findings are resolved, not before.
- Accepted deferred risks are documented with written rationale.
- Dated audits go stale. Absence of a recent audit does not mean the codebase is unreviewed — it means we haven't published one yet.
